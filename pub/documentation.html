Documentation

Getting Started

At first, you need to include this library and its style sheet in the Head part of your html file.

<script defer type="text/javascript" src='EasyLeaderboard.js'></script>
<link rel="stylesheet" type="text/css" href="./styles.css"></link>

To create a new Leaderboard, you need to have a div tag with a unique Id in the Body part of your html file.
Here is an example:

<div id="Example1"></div>

Constructors

createLeaderboard(querySelector, id, title, category, data)

This function will create a Default Leaderboard based on given parameters.
querySelector: String, the position you want to insert in html. In the previous example, "Example1" will be querySelector
id: String, unique id for this Leaderboard
title: String, title for this Leaderboard
category: Array includes String only, categories for the Leaderboard, which decides how many columns this Leaderboard will has
data: Array includes Dictionary only which Dictonaries must all have the same keys and these keys must be the same as category plus "id" (unique id), the content you want to display in Leaderboard

Example Code (See Examples page for actual performance):

const example1 = new EasyLeaderboard();
example1.createLeaderboard("Example1", "Example1", "SampleLeaderboard", ["Name", "Score"], [{"id": 1, "Name": "Gran", "Score": 10}, {"id": 2, "Name": "Djeeta", "Score": 20}, {"id": 3, "Name": "Lulia", "Score": 30}]);

Common Functions

insertRows(data)

This function will insert a new Row(s) in Leaderboard based on given data.
data: Array includes Dictionary only which Dictonaries must all have the same keys and these keys must be the same as category plus "id" (unique id)

Example Code (See Examples page for actual performance):

example1.insertRows([{"id": 4, "Name": "Vikala", "Score": 2020}, {"id": 5, "Name": "Catura", "Score": 2021}]);

deleteRow(id)

This function will delete a Row in Leaderboard based on given id.
id: Integer, unique id of row in Leaderboard, which is stored in dictionary mentioned above

Example Code (See Examples page for actual performance):

example1.deleteRow(2);

deleteCategory(category)

This function will delete a Category in Leaderboard based on given name.
Note: addCategory() is not provided since I recommend you to directly make a new Leaderboard to replace the old one if you want to add Category.
category: String, the name of target Category

Example Code (See Examples page for actual performance):

example1.deleteCategory("Name");

hideRow(id)

This function will hide (not delete) a Row based on given id.
id: Integer, unique id of row in Leaderboard, which is stored in dictionary mentioned above

Example Code (See Examples page for actual performance):

example2.hideRow(2)

showRow(id)

This function will show a hidden Row based on given id.
id: Integer, unique id of row in Leaderboard, which is stored in dictionary mentioned above

Example Code (See Examples page for actual performance):

example2.showRow(2)

hideCategory(category)

This function will hide (not delete) a Category based on given name.
category: String, the name of target Category

Example Code (See Examples page for actual performance):

example2.hideCategory("Score")

showCategory(category)

This function will show a hidden Category based on given name.
category: String, the name of target Category

Example Code (See Examples page for actual performance):

example2.showCategory("Score")

setMaximumRow(maxRow)

This function will set a limit on the maximum number of visible rows
category: Positive Integer, the maximum number of visible rows

Example Code (See Examples page for actual performance):

example2.setMaximumRow(3)

recoverLeaderboard()

This function for backup will delete the old Leaderboard and insert a new Leaderboard based on data stored in this Leaderboard Object, basically is used to show all hidden row and categories.
Example Code:

example1.recoverLeaderboard()
