Examples

1. Create a Default Leaderboard

Sample Code
const example1 = new EasyLeaderboard();
example1.createLeaderboard("Example1", "Example1", "SampleLeaderboard", ["Name", "Score"], [{"id": 1, "Name": "Gran", "Score": 10}, {"id": 2, "Name": "Djeeta", "Score": 20}, {"id": 3, "Name": "Lulia", "Score": 30}]);

2. Insert New Rows

Sample Code
example1.insertRows([{"id": 4, "Name": "Vikala", "Score": 2020}, {"id": 5, "Name": "Catura", "Score": 2021}]);

You can try this by yourself!

3. Delete a Row with given id

Sample Code
example1.deleteRow(2);

You can try this by yourself!

Note: The range of id in this example is 1 to 5. The row with id 2 has already been removed.

4. Delete a Category with given Name

Sample Code
example1.deleteCategory("Name");

5. Hide and Show a Row with given id

Sample Code
example2.hideRow(2)
example2.showRow(2)

You can try this by yourself!

Note: The range of id in this example is 1 to 5. The buttons above hide/show the row with id 2.

6. Hide a Category with given name

Sample Code
example2.hideCategory("Score")

You can try this by yourself!

Note: Ranking and Icon is not Category.

7. Set Maximum number of elements shown on a Leaderboard

Sample Code
example2.setMaximumRow(3)

You can try this by yourself!

Note: This example has only 5 rows.

8. Change User Icon with given id and src

Sample Code
example2.changeUserIcon(2, "test.jpg")

9. Set Link on User Icon with given id and url

Sample Code
example2.setLinkOnIcon(2, "https://easyleaderboard.herokuapp.com/documentation.html")
