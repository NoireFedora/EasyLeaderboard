EasyLeaderboard.js

This is a javascript library for you to easily make a simple customized Leaderboard based on your data on your webiste.

You are able to Insert/Remove, Hide/Show any rows by simple function calls. The Leaderboard has a basic sorting function based on given Category (Descending order for Number and Ascending order for Others). Removing/Hiding Category and Setting maximum number of elements shown on a Leaderboard is also possible. You are also able to Upload Icon and Set Link on Icon for each user.

For top 3 users, they will have special letter animation on their row. (First: gold, Second: silver, Third: bronze)

For details, please take a look on Examples and Documentation Sections.

I hope my library can make your programming life easier :)
